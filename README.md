# Simple Stopwatch Web App

A clean and responsive stopwatch web application built using **HTML**, **CSS**, and **JavaScript**. This project features start, pause, reset, and lap functionalities, providing an intuitive way to measure and record time intervals.

## Features

- Start/pause functionality with a single button
- Record lap times and view the list
- Reset to clear time and laps
- Responsive, modern UI
- Pure vanilla JavaScript (no frameworks)

## How to Use

1. **Clone the repository** or download the source files.

2. Open `index.html` in your favorite browser.

3. Use the buttons to start, pause, reset, and record lap times.
